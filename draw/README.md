# Confuse Matrix

The data in the file (STR-Transformer.txt) should be like this:
```
Label:0 --> Prob:0   # true label --> predict label
Label:0 --> Prob:3
...
```

Modify the path to your file (STR-Transformer.txt) in draw_cm.py and run it
